Question-1: What are the different ways to select an element in the DOM?

There are five ways to select element in the DOM.
These are getElementById, getElementsByClassName,getElementsByTagName,querySelector,querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML:

Returns the entire HTML structure of an element.

innerText:

Returns only the visible text content of an element, ignoring HTML tags and hidden elements.

textContent:

Returns all text content within an element, including hidden text, but without considering styling or HTML tags.

Question-3: What is event delegation in the DOM?

When we had many child element, as multiple li element in ul and we need to use event listner for each list.
Then we add event listner in parent tag. This process is known as event delegation.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating an Element

To create an element, we use the document.createElement() method.

Adding an Element

To add an element, we can use multiple method.
i.appendChild()
ii.append()
iii.prepend()
iv.insertBefore()

Removing an Element

To remove an element, we can use these method.
i.removeChild()
